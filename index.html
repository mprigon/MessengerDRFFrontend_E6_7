Механизм решения задания.

Frontend работает на Live Server, который запускается из VS Code

Адрес frontend сервера http://127.0.0.1:5500

Backend работает на сервере Django, который запускается из Pycharm.

Адрес backend сервера http://127.0.0.1:8000

API реализован на Django Rest Framework

  1. Сообщения. Посмотреть и отправить сообщение.
  2. Комнаты. Чат группы. Управление комнатами.
  3. Профиль пользователя. Изменить профиль.
  4. Вход в систему. Регистрация пользователя.
